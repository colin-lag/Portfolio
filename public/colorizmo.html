ROBO COLORIZMO

OVERVIEW

October - December 2019

Collaborator: Alexandre Berkovic

The task of this project was to create a robotic contraption which a user could interact with. My partner and I decided to create a 3-axis robotic gripping arm that could be controled by the user. When the controls and build were finished, we decided to create and implement a coloured cube sorting computer vision algorithm.
